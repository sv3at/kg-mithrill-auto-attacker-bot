Pre-requisites: 
* Python 3.12.4++
* pip install -r requirements.txt
* Run KG on a Desktop PC, vertically, and place it to the left of the screen
* Make sure all the troops besides the - most powerful one, are already gathering mithrill
* Make sure when you click on a mine, that one troop is selectable. 


Run: 
* Open advanced mithrill screen
* python .\attack.py
* Input your name, power & troop type (could be 123Fire) and nr. of remaining attack cycles.
* Watch it go through all the mines and save results into a CSV, including the success rates.
